# Changelog

All notable changes to NexusClaw are documented here.

## [0.22.0] — 2026-04-26

### Added
- Complete CLI with all commands (setup, chat, soul, tools, memory, kb, status)
- Comprehensive README with full feature matrix
- Final integration of all components

### Features
- FastAPI server (WebSocket + SSE streaming)
- OpenRoom web UI (dark theme, streaming, code highlighting, voice, file upload)
- Telegram + Discord channel bots
- Qdrant vector memory with semantic search
- Autonomy executor (goal planning, approval gates, kill switch)
- RAG pipeline (PDF/DOCX/HTML/TXT parsing + chunking + Q&A)
- Perplexity-style web search (Brave + DuckDuckGo)
- Code execution sandbox (Python/JS with limits)
- Knowledge base manager
- EvoClaw self-evolution engine (heartbeat, reflection, brain organs)
- Plugin system with hooks
- Tool registry with permission levels
- Soul editor (interactive identity builder)
- OpenClaw-style installer + docker-compose
- Full setup wizard (provider, memory, channels, soul)
